# Chat Challenge App 🚀

Bem-vindo ao repositório do projeto **Chat Challenge App**!

Este projeto é um *monorepo* (vários projetos contidos no mesmo repositório git) e consiste no desenvolvimento de uma aplicação de Chat completa. Ele está dividido em duas partes principais:

### 1. Aplicação Next.js (Front-end)
📁 **Diretório:** `chat-challenge-next-app`

A interface gráfica de usuário construída utilizando **Next.js**. Aqui se encontra a simulação do chat on-line entre usuários, com envio de novas instâncias de mensagens em tempo real e designs limpos.

### 2. Aplicação Node.js (Back-end)
📁 **Diretório:** `chat-challenge-node-app`

O servidor criado com **Node.js** e **Express**. Fornece a base para lidar com os WebSockets (ex: socket.io) para enviar, receber e retransmitir mensagens entre os clientes Next.js instantaneamente em tempo real.

---

## Como executar

Dependendo da parte do projeto que você estiver trabalhando, navegue até a pasta correta usando o terminal e rode os comandos de instalação e execução correspondentes (ex: `npm install` e, em seguida, `npm run dev`).
